# vim-config

--

### Theme

- Theme - molokai
- Font - monospace

### Tree file

```
.vim/
├── autoload/
    └── plug.vim
├── backup/
├── plugged/
├── colors/
    └── molokai.vim
```

### Install

```bash
cp -r vim ~/.vim
cp vimrc ~/.vimrc
```

### After install

command vim `:PlugInstall`

### Mapping

leader key - space

- CTRL + q - close file
- CTRL + s - save file
- SPACE + w - save file
- SPACE + e - open NerdTree files
